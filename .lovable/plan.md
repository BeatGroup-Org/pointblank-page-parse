

## Rimuovere le emoji dal messaggio WhatsApp

Modifica singola in `src/components/IscrizioneForm.tsx`: rimuovere tutti i prefissi emoji dalle righe del messaggio generato nel `handleSubmit`.

Da:
```
📋 ISCRIZIONE ...
👤 Nome: ...
📅 Nascita: ...
🏷️ CF: ...
📞 Tel: ...
📧 Email: ...
📍 Città: ...
🎵 Strumento: ...
📊 Livello: ...
✅ Partecipazione: ...
📝 Note: ...
```

A:
```
ISCRIZIONE ...

Nome: ...
Nascita: ...
CF: ...
Tel: ...
Email: ...
Città: ...
Strumento: ...
Livello: ...

Partecipazione: ...

Note: ...
```

Un solo file da modificare, nessun impatto su logica o validazione.
